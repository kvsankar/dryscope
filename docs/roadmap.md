# Roadmap

This document tracks planning notes that do not belong in the architecture
reference. Architecture describes how dryscope works today; this page describes
directions that may change.

## Future Directions

- Add parser and normalizer support for Rust and C/C++.
- Evaluate code-specific embedding models such as UniXcoder and CodeBERT.
- Keep improving benchmark labels and readable reports as public reference
  cases mature.
